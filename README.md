# Estadística Aplicada — Mediciones de Ruido Laboral

Análisis exploratorio y estadístico de mediciones de ruido en ambiente 
laboral. Aplica estadística descriptiva e inferencial para evaluar 
cumplimiento de límites legales y comparar zonas de trabajo.

## Qué resuelve
Un responsable de seguridad e higiene necesita saber qué zonas de la 
planta superan los límites legales de ruido y si hay diferencias 
estadísticamente significativas entre turnos o áreas.

## Datos
Dataset de muestra de mediciones de ruido (dB) por zona y turno.

## Qué incluye
- Estadística descriptiva (media, mediana, percentiles)
- Tests de normalidad (Shapiro-Wilk)
- Comparación entre grupos (t-test / Mann-Whitney)
- Intervalos de confianza
- Visualizaciones: boxplot, histograma, QQ-plot
